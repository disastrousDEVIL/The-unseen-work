# The Unseen Work

> Not all work is visible. This is the proof it happened.

## What is this?

I work as a Software Engineer specializing in  Agentic AI, Backend development, and DevOps across AWS and GCP. Most of my work happens in private repositories under my company's GitHub organization, which means my personal GitHub profile 
shows little to no activity — despite working every single day.

This repository exists to bridge that gap.

## The Problem

GitHub's contribution graph only shows activity from repositories tied to your 
personal account. When you work under a separate company GitHub account with 
private repositories, all of that work becomes invisible to the outside world.
Recruiters see an empty graph. The community sees an inactive developer.
Neither is true.

## How it works

A script runs automatically every day on my machine. It scans my work 
repositories, detects commits made under my work GitHub account, and mirrors 
them here as empty commits with matching timestamps.

- Zero code is ever exposed
- No proprietary information is shared
- Only the timestamps and frequency of work are reflected
